The following lists indicate the MIBs and SNMP notifications supported by Cisco Unified Computing System (UCS) Manager software releases 1.0 and later for the Cisco UCS 6100 series and Cisco UCS 6200 series. UCS Manager provides unified, embedded management of all software and hardware components of the Cisco UCS, including blade servers and rack servers.

·          In Release 1.0 and later, IETF networking MIBs such as the IF-MIB and the ENTITY-MIB are implemented.

·          In Release 1.3 and later, Cisco UCS can send SNMP traps whenever a fault is raised or cleared in Cisco UCS Manager.

·          In Release 1.4 and later, the entire UCS Manager data model is exposed through read-only UCS MIBs. All objects that can be retrieved through the Cisco UCS Manager XML API can also be retrieved through Cisco UCS Manager MIBs. The UCS Manager MIBs provide read-only access to all software and hardware components under the control of UCS Manager, including detailed information about managed blade servers and rack servers.

SNMP notifications can be enabled through GUI, CLI, or XML API. See the Cisco Configuration guides at the following website:

http://www.cisco.com/en/US/products/ps10281/tsd_products_support_series_home.html

For more details and up-to-date information, please see the MIB Quick Reference Guide for Cisco UCS:

http://www.cisco.com/en/US/docs/unified_computing/ucs/sw/mib/reference/UCS_MIBRef.html

All Cisco UCS MIBs are SNMP (SMIv2) MIBs.  Agent capabilities files are listed under their associated MIBs and contain implementation details on the MIB.  Agent capabilities files may contain multiple AGENT-CAPABILITIES macros. You must refer to the AGENT-CAPABILITIES macro that applies to your software release. Only SNMP GET requests are supported, SNMP SET requests are not supported.

Note: to obtain a list of MIBs supported by standalone UCS rack servers, see ftp://ftp.cisco.com/pub/mibs/supportlists/ucs/ucs-C-supportlist.html
