The following lists indicate the MIBs supported by Cisco Unified Computing System (UCS) C-Series CIMC SNMP Agent software releases 1.3 and later.

The UCS C-Series platforms supported by the SNMP software release:

1.     UCS C-22

2.     UCS C-24

3.     UCS C-200-M2

3.     UCS C-210-M2

3.     UCS C-260-M2

3.     UCS C-220-M3

4.     UCS C-240-M3

5.     UCS C-260

6.     UCS C-420

7.     UCS C-460

8.     UCS C-220-M4

9.     UCS C-240-M4

10.     UCS S3260

11.     UCS C-220-M5

12.     UCS C-240-M5

13.     UCS C-480-M5

A brief description of the SNMP agent releases are provided below, in the reverse-chronological order:

·         Release 2.0 Contains bug fixes for release 1.5.

·         Release 1.5 provides additional configuration options for snmp traps, com2sec settings, implementation for new snmp tables like Adaptor Unit Table, Rack Unit Table and contains bug fixes for release 1.4.

·         Release 1.4(3) provides SNMP V1/V2c/V3 support for inventory and fault monitoring of fans, power supplies, CPU, Memory and Storage Controllers, Physical Drives, Logical Drives and Battery Backup Units. The SNMP agent supports SNMP V1/V2c/V3 traps and V2c informs. This release also carries further the MIB-II Implementation, from the previous release, for “System”, “Interfaces” and “SNMP”.

·         Release 1.3(1) provides SNMP V1/V2c support for fault monitoring of fans and power supply along with V1/V2c trap support. This release also implements the MIB-II support for “System”, “Interfaces” and “SNMP”.

Note: Configuration Guides for SNMP parameters and traps using CLI and GUI may be found at the following location:

http://www.cisco.com/en/US/products/ps10739/products_installation_and_configuration_guides_list.html

For more details and up-to-date information, please see the MIB Quick Reference Guide for Cisco UCS:

http://www.cisco.com/en/US/docs/unified_computing/ucs/sw/mib/reference/UCS_MIBRef.html

All Cisco UCS MIBs are SNMP (SMIv2) MIBs. Only SNMP GET requests are supported, SNMP SET requests are not supported.

Note: to obtain a list of MIBs supported by UCS Manager managed servers, see ftp://ftp.cisco.com/pub/mibs/supportlists/ucs/ucs-manager-supportlist.html
