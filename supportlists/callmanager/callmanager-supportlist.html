MIBs supported by Cisco CallManager

The lists below indicate which MIBs are supported by the Cisco CallManager.

SNMP version 1 MIB files are in the v1 directory and SNMP version 2 MIB files are in the v2 directory. Note that for every MIB.my in the v2 directory, there exists a SNMP version 1 MIB-V1SMI.my  in the v1 directory as well.

     CISCO-CCM-MIB.my
     CISCO-CDP-MIB.my
     SYSAPPL-MIB.my
