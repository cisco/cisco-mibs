Cisco ASR 9000 Series SNMP MIB Support List

The following list indicates the MIBs that are supported by ASR 9000 Series Routers. SNMP version 1 MIBs are in the v1 directory and SNMP version 2 MIBs are in the v2 directory. Note that for almost every MIB .my file in the v2 directory, there exists a SNMP version 1 conversion of the MIB -V1SMI.my file in the v1 directory. MIBs are categorized into three types: Supported and Verified MIBs, Supported and Unverified MIBs, and Unsupported MIBs.
