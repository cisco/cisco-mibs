Cisco WAAS MIB Support List

The following lists indicate the MIBs and SNMP notifications supported by Cisco WAAS.

SNMP version 1 MIB files are in the v1 directory and SNMP version 2 MIB files are in the v2 directory. Note that for most every MIB in the v2 directory, there exists a SNMP version 1 conversion of the MIB in the v1 directory as well. All of these are SNMP (SMIv2) MIBs.

Agent Capabilities files are listed under their associated MIBs and contain implementation details on the MIB.  Agent Capabilities files may contain multiple AGENT-CAPABILITIES macros. You must refer to the  AGENT-CAPABILITIES macro that applies to your Cisco WAAS software release.

Last updated on 10-Dec-2012.
