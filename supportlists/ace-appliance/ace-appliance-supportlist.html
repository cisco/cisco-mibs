ACE4710 Application Control Engine Appliance

MIB Support List

The lists below indicates which MIBs and SNMP traps are supported by the Application Control Engine (ACE).

SNMP version 1 MIB files are in the v1 directory and SNMP version 2 MIB files are in the v2 directory. Note that for most every MIB in the v2 directory, there exists a SNMP version 1 conversion of the MIB in the v1 directory as well.

Note: 1. The maximum access provided by SNMP agent on ACE is read-only for all supported MIB objects.

          2. Additonal varbinds clogOriginIDType and clogOriginID as defined in CISCO-SYSLOG-MIB are appended to each SNMP notification. These can
              be used in recognizing the device from which the Notifications are generated.
